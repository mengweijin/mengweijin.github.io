    # 工具清单

    1. 8GB+ U盘一个
    2. UltraISO 软件。下载地址：https://www.ultraiso.com/ (opens new window)
    3. 系统映像：OpenEuler。下载地址：https://www.openeuler.org/zh/download/ (opens new window)
    4. 不用的 Windows 旧电脑一台

    注意：不要在太过老旧的电脑安装 OpenEuler，太过老旧的可以安装 CentOS 7。

    # 用 UltraISO 制作启动盘，刻录光盘映像

    • 将U盘插入电脑的USB接口
    • 用 UtralSO 打开已下载到本地硬盘的系统映像文件 openEuler-22.03-LTS-SP3-x86_64-dvd.iso
    • 单击UltraISO菜单栏中的 启动(B)，在弹出的下拉菜单中选择 写入硬盘映像
    • 在弹出的写入硬盘映像对话框中，确认 硬盘驱动器 栏中已正确识别出你的U盘型号后，单击 写入 按钮。
    • 在弹出的“提示”对话框中，单击 是 按钮，开始制作U盘引导安装盘（大概需要10分钟）。
    • 稍等片刻，即可在 UltraISO 的写入硬盘映像对话框中看到 刻录成功！ 的消息。

    # 修改 U 盘盘符名称

    用 UltraISO 刻录后的盘符名称默认应该是类似于这样的名称：openEuler-2，我们修改为 EULER。

    后面安装的时候要修改引导命令，否则安装操作系统的时候会报错。并且盘符名称需要为纯英文且长度不要超过 7 位。

    注意需要严格大小写，Windows 系统默认是大写，即使写小写也会转为大写，这里就直接用大写。

    # U 盘安装 OpenEuler 系统

    电脑插上 U 盘，然后开机，开机引导画面一出来，按 F12 进入 BIOS。（各个电脑品牌进入 BIOS 的按键不一样，根据自己的情况来。）

    然后选择 系统的加载顺序为从 U 盘启动加载，系统自动进入 GRUB 安装引导界面，如下：

    Install openEuler 22.03-LTS
Test this media & nstall openEuler 22.03-LTS

    这里不要直接安装，按 Tab 键，编辑启动命令。（有些电脑是按 E 键）

    vmlinuz initrd=initrd.img inst.stage2=hd:LABEL=openEuler-22.03-LTS-SP3-x86_64 rd.live.check quiet

# 修改 inst.stage2=hd:LABEL=盘符名称。这里是 EULER，如下：
vmlinuz initrd=initrd.img inst.stage2=hd:LABEL=EULER rd.live.check quiet
